# ImpactVenture – Digital Incubator IA + Green

## Overview
This project was developed as part of the Projet Technologies Web – 2nd Year 
Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).
ImpactVenture is a full-stack web platform that helps Tunisian entrepreneurs 
incubate their startup ideas with AI evaluation, green scoring, and mentor matching.

## Features
- AI-powered project evaluation (viability & innovation score)
- Green impact scoring (CO2 & social impact)
- Smart Theme Engine: dynamic theme creation + similarity scoring
- Mentor matching system
- Investment & financing module
- Training & badge system

## Tech Stack
### Frontend
- HTML5, JavaScript, Tailwind CSS (Front Office)
- AdminLTE + Bootstrap (Back Office)

### Backend
- PHP 8 (MVC architecture)
- MySQL with PDO

## Architecture
MVC Pattern — Controllers / Models / Views

## Getting Started
1. Clone the repository: `git clone https://github.com/[org]/Esprit-PW-2A23-2026-ImpactVenture.git`
2. Place in WAMP/XAMPP `www` folder
3. Import `database/impactventure.sql`
4. Access via `http://localhost/ImpactVenture`
